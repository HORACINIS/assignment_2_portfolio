El CARtel - Used cars marketplace

Website: https://elcartel.herokuapp.com/


Repository on GitHub: https://github.com/HORACINIS/el_carsales_marketplace


El CARtel, a full stack two-sided marketplace application with Ruby on Rails where a user can create an account and log into it using Devise.


Users can list their cars for sale and buy other users’ cars securely with the use of Stripe payment system. All user’s data, such as personal details and cars images uploaded are stored securely in AWS S3.
